# 比较

# setCompare 1.0.0+

setCompare([compareFunction])

# 描述

设置比较方法：为equal、greaterThan、lessThan、greaterThanOrEqual、lessThanOrEqual等API设置比较方法。如果不设置compareFunction则默认采用defaultcompare进行比较。

# 参数

  • compareFunction(Function)：可选，自定义比较方法

# 返回

(Object)：比较实例对象comparator

# 示例

import compartor from 'i-algorithms/_comparator'

comparator.setCompare()
comparator.greaterThan(1, 2)
// comparator.setCompare().greaterThan(1, 2)
// => false

comparator.setCompare((a, b) => {
  if (a === b) {
    return 0
  }
  return a > b ? 1 : -1
})
comparator.greaterThan(1, 2)
// => false

# defaultCompare 1.0.0+

setCompare(a, b)

# 描述

默认比较方法：equal、greaterThan、lessThan、greaterThanOrEqual、lessThanOrEqual等API的默认比较方法。

# 参数

  • a(Number)：比较元素
  • b(Number)：比较元素

# 返回

(Number)：如果a > b，返回1、如果a < b，则返回-1，否则返回0。

# 示例

import comparator from 'i-algorithms/_comparator'

comparator.defaultCompareFunction(1, 2)
// => -1

# equal 1.0.0+

equal(a, b)

# 描述

相等：判断两个元素是否相等。

# 参数

  • a(Any)：比较元素
  • b(Any)：比较元素

# 返回

(Boolean)：如果compareFunction返回0，则返回true，否则返回false。

# 示例

import comparator from 'i-algorithms/_comparator'

comparator.setCompare()
comparator.equal(1, 1)
// => true

comparator.setCompare((a, b) => {
  // 这里自定义两个数相等的规则，若a=b+1，则判定a=b
  if (a === b + 1) {
    return 0
  }

  return a > b ? 1 : -1
})

console.log(comparator.equal(2, 1))
// => true

# greaterThan 1.0.0+

greaterThan(a, b)

# 描述

大于：判断元素a是否大于元素b。

# 参数

  • a(Any)：比较元素
  • b(Any)：比较元素

# 返回

(Boolean)：如果compareFunction返回1，则返回true，否则返回false。

# 示例

import comparator from 'i-algorithms/_comparator'

comparator.setCompare()
comparator.greaterThan(2, 1)
// => true

# lessThan 1.0.0+

lessThan(a, b)

# 描述

小于：判断元素a是否小于元素b。

# 参数

  • a(Any)：比较元素
  • b(Any)：比较元素

# 返回

(Boolean)：如果compareFunction返回-1，则返回true，否则返回false。

# 示例

import comparator from 'i-algorithms/_comparator'

comparator.setCompare()
comparator.lessThan(2, 1)
// => false

# greaterThanOrEqual 1.0.0+

greaterThanOrEqual(a, b)

# 描述

大于或等于：判断元素a是否大于或等于元素b。

# 参数

  • a(Any)：比较元素
  • b(Any)：比较元素

# 返回

(Boolean)：如果compareFunction返回1或0，则返回true，否则返回false。

# 示例

import comparator from 'i-algorithms/_comparator'

comparator.setCompare()
comparator.greaterThanOrEqual(2, 1)
// => true

# lessThanOrEqual 1.0.0+

lessThanOrEqual(a, b)

# 描述

小于或等于：判断元素a是否小于或等于元素b。

# 参数

  • a(Any)：比较元素
  • b(Any)：比较元素

# 返回

(Boolean)：如果compareFunction返回-1或0，则返回true，否则返回false。

# 示例

import comparator from 'i-algorithms/_comparator'

comparator.setCompare()
comparator.lessThanOrEqual(2, 1)
// => false
