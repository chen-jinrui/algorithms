# 如何使用该仓库

# 🎉 克隆项目

克隆项目后切换到dev分支

git clone https://github.com/ziyi2/algorithms.git

# ➕ 安装依赖

npm install

# 🚨 运行ESLint

npm run lint

# ✅ 执行测试

会在coverage目录下生成测试覆盖率报告

npm test

# 🚧 启动静态网站

启动静态网站会开启ESLintr热校验

npm run dev

# 🔖 生成静态HTML文件

会在.vuepress/dist目录下生成静态资源包

npm run build

# 🚀 部署

生成静态HTML文件并更新到远程的gh-pages分支上

npm run deploy

# 📦 构建npm包

会在lib目录下生成平级目录结构的API使用包

npm run lib

# 🔊 生成日志

会在项目根目录下生成CHANGELOG.md日志文件

npm run changelog
